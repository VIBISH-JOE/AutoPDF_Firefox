# Auto PDF Downloader (Firefox)

Automatically captures and downloads protected PDFs from placement.skcet.ac.in, even when download is disabled. Opens the PDF and adds a "Download PDF" button at the bottom-right.

## Installation (Firefox)

1. Open Firefox
2. Go to: about:debugging#/runtime/this-firefox
3. Click "Load Temporary Add-on"
4. Select manifest.json from the extension folder

Now open any PDF on the site and click the "Download PDF" button in the bottom-right.
